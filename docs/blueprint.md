# **App Name**: ChronoBot Showcase

## Core Features:

- Interactive 3D Model: Display an interactive Three.JS model of the Discord Bot to visually engage users.
- Bot Functionality Showcase: This all-in-one Discord bot offers everything your server needs:

Aesthetic UI with Canvas-powered graphics
Advanced music system (Kazagumo + Shoukaku)
Anti-nuke and automod security
Powerful moderation and fun modules
Automation tools: auto-responder, auto-react, auto-pfp, voice roles, and more
Persistent data storage with MongoDB
Playlist-enabled welcome messages and profile cards
- Dynamic Command Reference: Display an organized and easily searchable command reference section.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) to represent modernity and activity.
- Background color: Deep Gray (#28282B) for a professional backdrop that helps content stand out.
- Accent color: Dark Purple (#4E2A84) to bring in the complementary color, suitable for highlights and CTAs.
- Body and headline font: 'Inter' for a modern, machined feel, ideal for a professional site showcasing bot functionalities.
- Use custom icons to represent the bot's functionalities and commands.
- Incorporate smooth transitions and subtle animations to enhance user experience.
- Maintain a clean and organized layout with clear sections and consistent design elements.
